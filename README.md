# 🏠 Property Management System - Innsight

## 📌 Overview
Innsight is a full-stack web application designed to help students and professionals find suitable accommodations like hostels, flats, and inns based on their preferences.

---

## 🚀 Features
- User Authentication (Login/Register)
- Property Search & Filtering
- Property Comparison
- Favorites Management
- Reviews & Ratings
- Owner Dashboard
- Booking & Offers System

---

## 🛠️ Tech Stack
**Frontend:**
- React.js
- Axios
- Bootstrap / CSS

**Backend:**
- Spring Boot
- REST APIs
- MySQL Database
  
---

## ⚙️ How to Run

### Backend
- cd backend
- mvn spring-boot:run

### Frontend
- cd frontend
- npm install
- npm start
